Features

Easy to Customize

Choose only the modules you need.

Secure by default

The security of the application was designed before the actual sharing features.

Super Fast

The source code is based on Rust and only a minimum of external crates are used.

Clean Design

Mycelium is designed to be as simple as possible to allow as much people as possible to review the source code and adapt features.

Design your own

If you don't like the offered frontend, just design your own. The whole source code is open source on GitHub.

Eco Friendly

Be part of the solution and reduce your carbon footprint by sharing goods with others.
